# Telegram-бот для учета заказов платков

Простой бот на `python-telegram-bot` с хранением данных в локальном `orders.json`.

## Возможности
- Кнопочное меню в Telegram:
  - `Новый заказ`
  - `Заказы за сегодня`
  - `Что отправить`
  - `Изменить статус`
  - `Отчет за сегодня`
  - `Отчет за месяц`
  - `Топ моделей за месяц`
- Поля заказа:
  - модель платка
  - количество
  - сумма в рублях
  - имя клиента
  - телефон
  - доставка (`СДЭК`, `Почта`, `Самовывоз`)
  - комментарий
  - статус
- Статусы:
  - `Новый`
  - `Подтвержден`
  - `Оплачен`
  - `Отправлен`
  - `Завершен`
  - `Отмена`
- Хранение заказов в `orders.json`

## Отчеты
### Отчет за сегодня
Показывает:
- количество заказов
- количество товаров
- общую сумму
- сколько оплачено
- сколько отправлено
- сколько отменено
- средний чек
- отдельные сводки по группам:
  - все заказы
  - оплаченные
  - отправленные

### Отчет за месяц
Показывает:
- количество заказов
- количество товаров
- общую сумму
- сколько оплачено
- сколько отправлено
- сколько завершено
- сколько отменено
- средний чек
- отдельные сводки по группам:
  - все заказы
  - оплаченные
  - отправленные

### Топ моделей за месяц
Показывает список моделей по количеству штук с сортировкой по убыванию отдельно для:
- всех заказов
- оплаченных
- отправленных

## Файлы проекта
- `main.py` — основной код бота
- `requirements.txt` — зависимости
- `.env.example` — пример переменных окружения
- `orders.json` — локальная база заказов

## Установка и запуск (Windows PowerShell)
1. Откройте PowerShell в папке проекта:
   ```powershell
   cd C:\Users\ice20\Desktop\bot
   ```
2. Создайте и активируйте виртуальное окружение:
   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   ```
3. Установите зависимости:
   ```powershell
   pip install -r requirements.txt
   ```
4. Создайте `.env` из шаблона:
   ```powershell
   Copy-Item .env.example .env
   ```
5. Откройте `.env` и вставьте свои значения:
   ```env
   BOT_TOKEN=ваш_токен_бота
   ADMIN_ID=ваш_числовой_telegram_id
   ```
6. Запустите бота:
   ```powershell
   python .\main.py
   ```

## Где взять данные
- `BOT_TOKEN`: создайте бота у [@BotFather](https://t.me/BotFather) и возьмите токен.
- `ADMIN_ID`: ваш числовой Telegram ID (например, через ботов типа `@userinfobot`).

## Использование
1. Напишите боту `/start`.
2. Выбирайте действия кнопками в главном меню.
3. Для отмены текущего шага можно отправить `/cancel`.

## Примечания
- Пункт `Что отправить` показывает заказы со статусом `Оплачен`.
- Период `за месяц` считается по текущему календарному месяцу.
- Если `ADMIN_ID` не указан как число, бот не ограничивает доступ по ID.
